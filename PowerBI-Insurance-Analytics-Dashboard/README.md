# Insurance Analytics Dashboard (Power BI)

## Project Overview
Interactive Insurance Analytics Dashboard developed using Power BI to analyze policy performance, claim trends, customer distribution, and insurance coverage metrics.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Key Features
- Premium Amount Analysis
- Coverage Amount Tracking
- Claim Amount Monitoring
- Policy Type Analysis
- Active vs Inactive Policies
- Claim Status Analysis
- Age Group Analysis
- Interactive Filters

## KPIs
- Premium Amount
- Coverage Amount
- Claim Amount

## Skills Demonstrated
- Data Cleaning
- Data Modeling
- DAX Calculations
- Dashboard Design
- Business Intelligence
