# Sales Analysis Dashboard (Power BI)

## Overview
Interactive Power BI dashboard created to analyze sales performance, profit trends, product performance and city-wise sales distribution.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Dashboard Features
- Sales by City Map
- Profit vs Net Sales Analysis
- Sales Trend Analysis
- Top 5 Products Analysis
- Bottom 5 Products Analysis
- Interactive Filters
- Detailed Sales Table

## KPIs
- Total Sales
- Total Profit
- Total Quantity Sold

## Skills Demonstrated
- Data Cleaning
- Data Visualization
- DAX Calculations
- Dashboard Design
- Business Analysis
