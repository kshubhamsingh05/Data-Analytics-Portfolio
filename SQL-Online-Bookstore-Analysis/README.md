# Online Bookstore SQL Analysis

## Project Overview
This project analyzes an Online Bookstore database using SQL Server. The goal is to extract meaningful business insights by writing SQL queries.

## Tools Used
- SQL Server
- SQL

## Skills Demonstrated
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate Functions (SUM, COUNT, AVG)
-  JOIN
- Sorting and Filtering

## Database Tables
- Books
- Customers
- Orders

## Business Questions Solved
- Total books sold by each author
- Customer who spent the most
- Cities where customers spent more than $30
- Total sales analysis
- Author-wise book sales
- Customer order analysis

## Key Learnings
- Writing SQL queries
- Working with multiple tables
- Using JOINs and Aggregate Functions
- Analyzing sales data
